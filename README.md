## https://go-tour-jp.appspot.com/basics/7
- GOってなに？

- ステートメントってなに？
  - たいてい、他のワードと組み合わせて書いてあるので、ほかのワードそのものを指す。
  - 例えば return は、原文の英語だと return だけだと意味が通じなくなるのでステートメントってつける。
